Todo Application
A console-based todo application built with Python.

Features
Add new todo items
View all todo items
Update existing todo items
Mark todo items as complete/incomplete
Delete todo items
Setup
Ensure you have Python 3.13+ installed
Install dependencies: uv sync (if using uv)
Run the application: python -m src.todo_app.main
Usage
Run the application with various commands:

add "title" "description" - Add a new todo
list - List all todos
update id "title" "description" - Update a todo
complete id - Mark a todo as complete
incomplete id - Mark a todo as incomplete
delete id - Delete a todo
